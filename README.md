
# Dart Projects Repository

This repository contains my Dart projects for learning Classes, Objects, Functional Programming, and OOP concepts.

---

##  Project Files

1. `Milestone1_YourName.dart`  
   - Grade Calculator project  
   - Demonstrates functions, collections, and higher-order functions in Dart.

2. `User.dart`  
   - Immutable data class example  
   - Includes copyWith, toString, equality (`==`), and hashCode.

3. `AdvancedClasses.dart`  
   - Advanced Dart OOP examples  
   - Demonstrates inheritance, polymorphism, abstract classes, and interfaces.

4. `GradeCalculator_UML.pdf`  
   - UML class diagram showing class relationships and project structure.

---

## How to Run the Code

1. Install Dart: [https://dart.dev/get-dart](https://dart.dev/get-dart)  
2. Open terminal or command prompt in the project folder.  
3. Run any Dart file:

```bash
dart Milestone1_YourName.dart
dart User.dart
dart AdvancedClasses.dart
