# Dart Projects Repository – Milestone & Exercises

This repository contains my Dart projects demonstrating Classes, Objects, OOP, and Functional Programming concepts. It includes my main milestone project, UML diagrams, user guide, and additional exercises.  

---

##  Project Structure

### Main Milestone – Grade Calculator
- `Milestone1_YourName.dart` – Grade Calculator project demonstrating functions, collections, and higher-order functions.
- `User.dart` – Immutable data class with copyWith, toString, equality (`==`), and hashCode.
- `AdvancedClasses.dart` – Advanced Dart OOP examples: inheritance, polymorphism, abstract classes, interfaces.
- `GradeCalculator_UML.pdf` – UML diagram showing class relationships for Grade Calculator.
- `AdvancedClasses_UML.pdf` – UML diagram for Advanced Classes.
- `User Guide.pdf` – Step-by-step guide for using the Grade Calculator.

---

### Exercises Folder
Contains additional exercises I completed in class:

- `ZooAnimal.dart` – Demonstrates inheritance and polymorphism with animals.  
- `StudentOOP.dart` – Shows data classes, class methods, and object interactions.  
- `PetShop.dart` – Example of nested and inner classes, showcasing class relationships.

> Optionally, you can create a folder /Exercises and place all exercise files there to keep the repo organized.

---

##  How to Run the Dart Files

1. Install Dart: [https://dart.dev/get-dart](https://dart.dev/get-dart)  
2. Open terminal/command prompt in the repository folder.  
3. Run a Dart file using:

```bash
dart Milestone1_YourName.dart
dart User.dart
dart AdvancedClasses.dart
dart ZooAnimal.dart
dart StudentOOP.dart
dart PetShop.dart
